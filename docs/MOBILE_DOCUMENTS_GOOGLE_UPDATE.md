# Mobile, Documents, Pagination and Google Login Update

- Responsive hardening for phone, tablet and desktop layouts.
- Trucker documents now have separate Required and Other Documents sections.
- Other documents support unlimited uploads.
- Dispatcher can upload both required and other documents for assigned truckers.
- Document lists use database pagination with numbered controls.
- React 19 form warnings were removed by allowing function actions to provide form encoding automatically.
- Google OAuth supports explicit redirect URI configuration and account linking.
